# Cerebral-Stroke-Prediction-Using-Random-Forest

This Project was made using a dataset that includes information about patients and whether they might get a cerebral stroke or not,

Features included in this dataset were: Gender, Age, Hypertension, Heart disease history, Marriage status, Work type, Residence, Average Glucose levels, BMI, and Smoking status.

This dataset was first preprocessed by checking for redundant and missing values, then visualized to show the imbalance in the dataset.

Then it was split into a 80% train/ 20% test split to create a model to predict cerebral stroke.

Some of the methods used in this model were: Standarization (Z-Score), SMOTE (Synthetic Minority Over-sampling Technique), and Random Under Sampling to balance the dataset.

Some other models were used to see which one would give the best results, such as, Logistic Regression, SVM, XgBoost, and Random Forest Classifier.
